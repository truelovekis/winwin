### ⚙️ 개발 환경
- **Back-end** : `Java 8`
- **Framework** : Ajax, MyBatis, Spring Boot, thymeleaf
- **Database** : Oracle
- **ORM** : Mybatis
- **Front-End** : HTML, CSS, JavaScript
- **Devops** : GitHub
- **API** : SMS API
